# GitHub Repository Metadata

Use these values in the GitHub repository settings to present the project as a
portfolio-ready infrastructure repository.

## Recommended Repository Name

`llm-serving-gateway-vllm`

This name is clearer than `Mini_LLM_Serving_Platform` because it leads with the
engineering concept recruiters and engineers will search for: an LLM serving
gateway backed by vLLM.

## Recommended Display Title

OpenAI-Compatible LLM Serving Gateway

## Recommended About Description

Production-style OpenAI-compatible LLM serving gateway with vLLM, Redis quotas,
Prometheus/Grafana, benchmarks, Docker/K8s/Helm, GitOps, Terraform, and CI/CD.

## Recommended Website Field
